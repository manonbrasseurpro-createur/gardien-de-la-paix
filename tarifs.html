Petits tests vs tests complets

Petits tests (1 essai gratuit sans abonnement) : mini tests cognitifs, mini culture/langue, mini test culture générale, entraînement par catégorie (10 questions), question isolée en cas pratiques.

Tests complets (abonnement requis) : simulations alignées sur le concours externe — psychotechniques ~2 h (non notés au concours), culture + langue ~1 h, cas pratiques ~2 h (mode entraînement 3 cas), personnalité 60 affirmations, entraînements verbal/spatial/numérique, page Ma progression.

Paiement via Stripe. Après paiement, votre compte est activé automatiquement (configuration Supabase + webhook Stripe requise — voir README).
