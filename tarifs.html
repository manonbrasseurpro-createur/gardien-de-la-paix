Petits tests vs tests complets

Petits tests (1 essai gratuit sans abonnement) : mini tests cognitifs, mini culture/langue, mini test culture générale, entraînement par catégorie (10 questions), question isolée en cas pratiques.

Tests complets (abonnement requis) : simulations alignées sur le concours externe — psychotechniques ~2 h, culture + langue ~1 h, cas pratiques ~2 h, personnalité 60 affirmations, entraînements verbal/spatial/numérique, page Ma progression.

Après paiement, vous êtes redirigé vers une page de confirmation. L'abonnement est activé automatiquement via Stripe.
